2026-04-13 · 动态规划

01

背包问题：从 0-1 背包到完全背包

0-1 背包的核心状态是 dp[j] 表示容量为 j 时的最大价值。每个物品只能选一次，所以容量要倒序遍历： dp[j] = max(dp[j], dp[j - w[i]] + v[i])。完全背包中物品可以重复选择，容量改为正序遍历。 

for each item i:
  for j from capacity down to weight[i]:
    dp[j] = max(dp[j], dp[j - weight[i]] + value[i])

2026-04-14 · 双指针

02

接雨水：为什么双指针能做到 O(n)

对每个位置，能接的水由左右两侧最高柱子的较小值决定。双指针维护 leftMax 和 rightMax，每次移动较低的一侧，因为这一侧的上界已经确定，可以立即计算当前位置的积水。 

while left < right:
  if height[left] < height[right]:
    update leftMax and answer
    left += 1
  else:
    update rightMax and answer
    right -= 1

2026-05-11 · 状压 DP

03

旅行商问题 TSP：状态压缩动态规划入门

TSP 要求从起点出发访问所有城市并回到起点。经典做法是用二进制集合表示已访问城市： dp[mask][i] 表示访问集合为 mask 且最后停在城市 i 的最短路径。枚举下一座未访问城市转移。 

dp[1 << start][start] = 0
for mask in all states:
  for i in cities:
    for j not in mask:
      dp[mask | (1 << j)][j] =
        min(dp[mask | (1 << j)][j], dp[mask][i] + dist[i][j])

2026-05-08 · 图论

04

Dijkstra：单源最短路的优先队列写法

当边权非负时，Dijkstra 每次取出当前距离最小的点，并尝试松弛它的邻边。使用优先队列后复杂度通常为 O((n + m) log n)，适合稀疏图。 

dist[start] = 0
push (0, start)
while heap is not empty:
  d, u = pop min
  if d > dist[u]: continue
  relax all edges from u

2026-05-06 · 动态规划

05

最长公共子序列 LCS：二维 DP 的边界意识

LCS 的状态 dp[i][j] 表示字符串 A 前 i 个字符与字符串 B 前 j 个字符的最长公共子序列长度。字符相等时来自左上角加一，否则取上方和左方最大值。 

if a[i - 1] == b[j - 1]:
  dp[i][j] = dp[i - 1][j - 1] + 1
else:
  dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
