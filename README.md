# 个人主页模板

这是一个适合保研申请、夏令营申请和学术展示的 GitHub Pages 静态个人主页模板。

## 使用方式

1. 把本文件夹内容上传到你的 GitHub 仓库：`你的用户名.github.io`
2. 将 `index.html` 里的姓名、学校、GPA、科研经历、项目和邮箱替换成自己的信息
3. 把你的简历 PDF 命名为 `resume.pdf`，放在项目根目录
4. 在 GitHub 仓库的 `Settings > Pages` 中选择 `main` 分支发布

上线后访问：

```text
https://你的用户名.github.io
```

## 文件说明

- `index.html`：网站内容
- `styles.css`：视觉样式
- `script.js`：导航高亮
- `assets/hero-academic.png`：首页背景图
- `resume.pdf`：你的简历文件，需要自行添加
