旅行商问题 TSP：状态压缩动态规划入门

TSP 是理解状态压缩动态规划的经典例题。

问题描述

从起点出发，访问每座城市恰好一次，最后回到起点，要求总路程最短。暴力枚举排列的复杂度是阶乘级，城市数量稍大就不可接受。

状态设计

用二进制整数 mask 表示已经访问过的城市集合。dp[mask][i] 表示当前访问集合为 mask，最后停在城市 i 的最短距离。

INF = 10 ** 18
dp = [[INF] * n for _ in range(1 << n)]
dp[1][0] = 0

for mask in range(1 << n):
    for i in range(n):
        if dp[mask][i] == INF:
            continue
        for j in range(n):
            if mask & (1 << j):
                continue
            nxt = mask | (1 << j)
            dp[nxt][j] = min(dp[nxt][j], dp[mask][i] + dist[i][j])

ans = min(dp[(1 << n) - 1][i] + dist[i][0] for i in range(n))

复杂度

状态数是 2^n * n，转移还要枚举下一座城市，所以时间复杂度约为 O(2^n * n^2)，适用于城市数量较小的场景。

学习心得

状态压缩最重要的是把“集合”变成一个可以枚举和转移的整数。TSP 让我更直观地理解了为什么有些指数级问题仍然可以通过 DP 在小规模输入下求出精确解。
