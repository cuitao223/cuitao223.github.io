课程总结 · 线性代数

线性代数知识点：行列式、矩阵、向量组、特征值与二次型

这篇笔记按知识点拆分线性代数，重点记录常考概念、计算方法和容易混淆的判断条件。

一、行列式

  • 定义与性质：互换两行变号，某行乘常数则行列式乘该常数，某行加另一行倍数行列式不变。
  • 展开定理：按行或列展开，代数余子式符号为 (-1)^(i+j)。
  • 常见计算：三角化、按零多的行列展开、利用范德蒙德行列式、分块矩阵行列式。
  • 重要结论：|AB| = |A||B|，|A^T| = |A|，可逆矩阵行列式非零。

二、矩阵

  • 基本运算：矩阵加法、数乘、乘法、转置、逆矩阵、伴随矩阵。
  • 矩阵秩：行秩等于列秩；初等变换不改变秩；r(A) 反映矩阵中独立信息量。
  • 逆矩阵判断：方阵可逆等价于行列式非零、满秩、齐次方程只有零解。
  • 初等矩阵：左乘初等矩阵对应行变换，右乘初等矩阵对应列变换。
  • 分块矩阵：适合处理高维矩阵乘法、逆矩阵和行列式计算。

三、线性方程组

  • 齐次方程组：Ax = 0，非零解存在条件是 r(A) < n。
  • 非齐次方程组：Ax = b，有解条件是 r(A) = r(A|b)。
  • 解的结构：非齐次通解 = 一个特解 + 对应齐次方程组通解。
  • 基础解系：自由变量个数为 n - r(A)，也是解空间维数。

四、向量组与向量空间

  • 线性相关与无关：一个向量能由其余向量线性表示，则向量组线性相关。
  • 极大无关组：能代表原向量组张成空间的最小独立骨架。
  • 等价向量组：可以相互线性表示，秩相同，张成空间相同。
  • 基与维数：基是一组线性无关且能张成空间的向量，维数是基中向量个数。

五、特征值、特征向量与相似对角化

  • 特征值定义：满足 Ax = lambda x 的非零向量 x 是特征向量。
  • 求解步骤：先解 |lambda I - A| = 0 得特征值，再解 (lambda I - A)x = 0 得特征向量。
  • 相似矩阵：B = P^{-1}AP，相似矩阵有相同特征值、行列式和迹。
  • 可对角化条件：有 n 个线性无关特征向量；不同特征值对应特征向量线性无关。
  • 实对称矩阵：一定可以正交对角化，不同特征值对应特征向量正交。

六、二次型

  • 矩阵表示：二次型可写成 x^T A x，其中 A 通常取实对称矩阵。
  • 标准形与规范形：通过合同变换化简，关注平方项系数。
  • 正定判断：顺序主子式全为正，或特征值全为正。
  • 惯性定理：正负惯性指数在合同变换下不变。

复习心得

线代复习要把“秩”作为核心线索。行列式、方程组、向量组、可逆性和特征值很多判断都能回到秩、线性无关和空间维数上。
