最长公共子序列 LCS：二维 DP 的边界意识

LCS 是练习二维 DP 定义和初始化的基础题。

状态定义

dp[i][j] 表示字符串 a 的前 i 个字符和字符串 b 的前 j 个字符的最长公共子序列长度。

状态转移

如果末尾字符相同，说明这个字符可以接在前一个公共子序列后面；否则只能从去掉其中一个末尾字符的两个状态中取最大值。

for i in range(1, len(a) + 1):
    for j in range(1, len(b) + 1):
        if a[i - 1] == b[j - 1]:
            dp[i][j] = dp[i - 1][j - 1] + 1
        else:
            dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])

边界处理

把 dp 开成 (len(a) + 1) * (len(b) + 1)，第 0 行和第 0 列表示空字符串，可以自然处理边界。

学习心得

LCS 让我意识到 DP 状态不要急着从下标 0 开始硬凑。多开一行一列虽然看起来浪费空间，但能让转移表达更清楚，代码也更不容易错。
