Dijkstra：单源最短路的优先队列写法

Dijkstra 用于求非负边权图上的单源最短路径。

算法思想

维护起点到每个点的当前最短距离。每次从优先队列中取出距离最小的点，并用它去松弛相邻边。因为边权非负，被取出的最短距离不会再被更小路径推翻。

import heapq

dist = [float("inf")] * n
dist[start] = 0
heap = [(0, start)]

while heap:
    d, u = heapq.heappop(heap)
    if d > dist[u]:
        continue
    for v, w in graph[u]:
        if dist[v] > dist[u] + w:
            dist[v] = dist[u] + w
            heapq.heappush(heap, (dist[v], v))

注意点

  • 不能处理负权边。
  • 优先队列里可能存在旧状态，取出后需要判断 d > dist[u]。
  • 稀疏图中邻接表比邻接矩阵更合适。

学习心得

我以前容易把 Dijkstra 记成固定模板，但真正关键的是“每次确定一个当前最短点”。只要抓住非负边权这个前提，优先队列写法的正确性就更容易理解。
